# ✅ To-Do List App

A simple **To-Do List web application** built with pure **HTML, CSS, and JavaScript**. The project allows users to add, complete, and delete tasks through a clean and minimal interface, designed as a personal practice project for front-end fundamentals and DOM manipulation.

🔗 **Live Demo:** [to-do-list-seven-theta-97.vercel.app](https://to-do-list-seven-theta-97.vercel.app)

## ✨ Features

- 📝 **Add Tasks** — Quickly add new items to your to-do list
- ✅ **Mark as Complete** — Check off tasks once finished
- 🗑️ **Delete Tasks** — Remove tasks that are no longer needed
- ⚡ **Real-Time UI Updates** — List updates instantly via DOM manipulation
- 🎨 **Simple, Clean Interface** — Minimal design with no external frameworks

## 🚀 Getting Started

### Prerequisites

- A modern web browser
- (Optional) [Git](https://git-scm.com/) for cloning the repository

### 1. Clone the Repository

```
git clone https://github.com/Pongpun1/To-Do-List.git
cd To-Do-List
```

### 2. Run the Application

Simply open `index.html` in your browser:

```
open index.html
```

No build steps, dependencies, or server setup required.
